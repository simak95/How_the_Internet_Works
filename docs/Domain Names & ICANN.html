Domain Names and ICANN

Domain Names

A domain name serves as an address that is used as a unique identifier for websites. The domain name of a website is visible on the address bar where it generally begins with 'www' and ends with a suffix such as '.com,' '.org,' '.net.' Using the Domain Name System, the domain names get translated to IP addresses that allow users to access websites simply using the domain name. Nowadays, registering for domain names for your website is relatively easy and affordable with only the risk of having duplicates given that commonly used terms have already been registered for use. A good example of a domain name is www.google.com, a site crucial to gathering knowledge in the 21st century.

ICANN

Internet Corporation For Assigned Names and Numbers (ICANN), an American non-profit organization that is responsible for allocating IP Addresses and managing the Domain Name System. ICANN allocates blocks of IP Addresses to various companies, institutions and internet service providers, where the companies decide to allocate unique IP addresses rather than ICANN allocating IP addresses individually. The organization aims to preserve the Internet's operational stability, promote competition among websites, achieve broad global representation of Internet communities among many other long term goals to ensure the integrity of the Internet. ICANN is governed by an international Board of Directors that overseas the development process.
